## Hardware Abstraction Layer (HAL)

Cube HAL provides the confidential-computing runtime environment for Cube AI.

## Steps

1. Clone repositories  
2. Configure Buildroot  
3. Build the image  
4. Boot inside an SEV-SNP CVM  
5. Run the Cube AI stack
